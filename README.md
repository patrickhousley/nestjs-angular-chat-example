# Nestjs Angular Chat Example

Welcome to the companion repository for my Medium series on combining NestJS and Angular 6 in a monorepo-styled project. This project makes use of Nrwl Nx to create and maintain a well structured monorepo-styled project. Each article in the series is accompanied by it's own branch.

- [Chatting it up with NestJS and Angular 6: Setting up the Project](https://medium.com/@patrickhousley/chatting-it-up-with-nestjs-and-angular-6-setting-up-the-project-c3aa24674f9b)
  - [Branch: part-01](https://github.com/patrickhousley/nestjs-angular-chat-example/tree/part-01)
